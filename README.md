Academia PowerFit
Um programa de academia ╰（‵□′）╯

  Funcionalidades:

- Cadastro de alunos.
- Cadastro de treinos de:
  - Força
  - Cardio
  - Flexibilidade
- Cálculo de calorias para cada tipo de treino.
- Cálculo do ganho de mobilidade em treinos de flexibilidade.
- Plano semanal contendo vários tipos de treino.
- Soma do total de calorias gastas durante a semana.
- Restrição automática para alunos iniciantes realizarem treinos acima do seu nível.

O sistema segue com as seguintes mecânicas:

  O cálculo de calorias varia conforme o tipo de treino:
1° - Força: baseado no peso levantado e número de séries.
2° - Cardio: baseado no tempo e intensidade.
3° - Flexibilidade: baseado apenas no tempo, além de falar o ganho de mobilidade.
